public function SMTPErrorEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, text:String = "", replyCode:int = 0)

Creates a new SMTPErrorEvent object initialized with the specified type, description and reply code.

Parameters
type:String — The type of the event, accessible in the type property. The SMTPErrorEvent defines one event type, the smtpError event, represented by the SMTPErrorEvent.SMTP_ERROR constant.
 
bubbles:Boolean (default = false) — Determines whether the event object participates in the bubbling stage of the event flow. The default value is false.
 
cancelable:Boolean (default = false) — Determines whether the event object can be cancelled. The default value is false.
 
text:String (default = "") — Text to be displayed as an error message. Event listeners can access this information through the text property.
 
replyCode:int (default = 0) — The reply code as returned by the SMTP service.
